练习 1.42
================

对于表达式：

::

    ((compose square 1+) 6)

要计算出值 ``49`` ，这个表达式应该执行以下计算序列：

::

    ((compose square 1+) 6)

    (square (1+ 6))

也即是， ``(compose square 1+)`` 的展开式应该为：

::

    (lambda (x)
        (square (1+ x)))
